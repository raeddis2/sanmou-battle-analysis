# 普攻伤害公式推导

日期：2026-06-22

## 当前结论

基于当前 SQLite `data/sanmou_battles.sqlite` 中 1095 条普通攻击伤害记录，剥离兵力、兵种克制、显式增减伤、国家条件增减伤、品级隐式项和会心后，普通攻击的基础攻防项最稳定的形式是：

```text
基础普攻伤害 ≈ 287 × exp(0.0030 × (出手时武力 - 目标出手时统率))
```

因此当前完整口径可写为：

```text
普攻伤害 ≈ round(
    287
  × exp(0.0030 × (A_force_now - D_command_now))
  × F_troop(A_hp_now)
  × M_unit_counter
  × M_attacker_damage
  × M_defender_taken
  × M_grade
  × M_special
)
```

其中：

```text
F_troop(N) = 1                    , N >= 9000
F_troop(N) = (N / 9000) ^ 0.38     , N < 9000
```

## 乘区定义

- `A_force_now`：本次出手时攻击方实时武力，读取 `damage_contexts.source_force`。
- `D_command_now`：本次受击时目标实时统率，读取 `damage_contexts.target_command`。
- `A_hp_now`：攻击方当前兵力，优先读取 `source_context_json.hp`，为空时用 `participants.initial_troops`。
- `M_unit_counter`：兵种克制独立乘区，攻方克制目标 `1.15`，被克制 `0.85`，无克制 `1.00`。
- `M_attacker_damage`：攻方加法增伤桶，临时口径为 `1 + (source_damage_pct + 对目标国家武将伤害提升) / 100`。
- `M_defender_taken`：目标受伤桶，临时口径为 `1 + (target_damage_taken_pct - 受到攻方国家武将伤害降低) / 100`。
- `M_grade`：品级隐式项，临时口径为 `(1 + 攻方品级/100) × (1 - 目标品级/100)`；NPC 强制按 0 品处理。
- `M_special`：本次特殊乘区。会心/奇谋按日志里的 `recent_trigger.damage_ratio` 读取，常见 `1.5` 或 `1.6`；无触发为 `1`。抵御、云身、协防/承担、完璧、虚弱等仍需单独建模或剔除污染样本。

## 场外属性因子

本轮把场外武力/统率与实时武力/统率同时放进模型后，场外项几乎没有独立解释力：

```text
log(伤害/其他乘区) = 5.6583
                 + 0.00016 × (场外武力 - 场外统率)
                 + 0.00297 × (实时武力 - 实时统率)
```

在只保留实时差值后，残差与场外差值的相关性约 `0.028`。所以目前不建议把场外属性作为独立乘区；更稳的理解是：场外属性通过战斗内属性变化进入 `A_force_now` 和 `D_command_now`，也就是已经被实时攻防项吸收。

如果必须按“场外属性因子 × 出手时攻防因子”拆写，可写成近似分解：

```text
M_off_attr ≈ exp(0.00016 × (A_force_off - D_command_off))  # 当前近似为 1
M_live_attr ≈ exp(0.00297 × (A_force_now - D_command_now))
```

但当前证据支持把 `M_off_attr` 置为 `1`，直接使用实时攻防差。

## 拟合表现

样本口径：

- 普攻样本：1095 条。
- 可建模样本：1068 条。
- 无当前会心/奇谋样本：894 条。
- 会心样本按日志倍率放回后：174 条。

用无触发样本拟合、再评估：

```text
无触发样本：rmse_log=0.1172，平均相对误差约 7.6%，中位相对误差约 5.3%
会心样本：rmse_log=0.0884，平均相对误差约 5.5%，中位相对误差约 4.1%
合并样本：rmse_log=0.1130，平均相对误差约 7.3%，中位相对误差约 5.1%
```

会心样本用无触发公式反推倍率，中位约 `1.55`，与日志中 `150%/160%` 两档一致。

## 注意事项

1. 当前常数 `287` 和系数 `0.0030` 是当前库样本的拟合中心，不是最终官式。
2. 低兵力极端样本、吕蒙低兵力普攻、马云禄打马超等少数样本残差较大，可能还有未剥离的特殊机制或取数污染。
3. `target_hp_before` 不能作为防守方兵力因子纳入当前公式；此前拟合防守方兵力解释力弱。
4. 不要把 `damage_contexts.source_force/target_command` 称为场外属性，它们是实时属性快照。
